# Liberatus-T480
> *Engineering repository for the Lenovo Thinkpad T480 open-source firmware implementation and hardware extensibility.*
## Bios Switcher
A 2 NOR chip switcher allowing you to effectively dual-boot firmware. Useful for firmware developers or overclockers. 
## EGPU Harness
A external PCI-E card extension using the MX150 BGA available PCI-E lanes. Useful for gaming, data science or LLMs.
